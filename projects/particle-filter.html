Overview

Implemented a Monte Carlo Localization (MCL) system using particle filtering for a mobile robot operating on the SE(2) manifold (2D position + orientation). The filter maintains a set of weighted particles representing the belief distribution over robot pose, updating them with motion and sensor models at each timestep.

Problem

Global localization ‚Äî determining a robot's pose in a known map without prior knowledge of its starting position ‚Äî is a challenging problem because the initial belief is spread across the entire configuration space. Unlike Gaussian filters like the EKF, particle filters can represent multimodal distributions, making them well-suited for this "kidnapped robot" scenario.

Approach

The particle filter follows the standard predict-update cycle. In the prediction step, each particle is propagated forward using a probabilistic motion model that accounts for odometry noise. In the update step, sensor measurements are used to compute importance weights for each particle based on their likelihood.

Systematic resampling is applied to focus computational resources on high-probability regions of the state space. Special care was taken to handle the topology of SE(2), where the orientation component wraps around at ¬±œÄ, requiring proper angular arithmetic throughout the filter.

Key Features

SE(2) Manifold Handling

Proper treatment of orientation wrapping and angular statistics on the SE(2) group.

Multimodal Belief

Particle representation naturally captures multiple hypotheses during global localization.

Probabilistic Motion Model

Odometry-based motion model with noise parameters tuned to real sensor characteristics.

Systematic Resampling

Low-variance resampling to prevent particle depletion while maintaining diversity.

Results

The particle filter successfully converged from a uniform prior to the correct robot pose within a small number of observation steps. The implementation demonstrated robust global localization and recovery from simulated kidnapping events, with particle clouds visibly converging as landmark observations accumulated.

What I Learned

This project solidified my understanding of non-parametric Bayesian estimation and the practical tradeoffs in particle filter design ‚Äî particle count vs. computational cost, resampling strategies, and the importance of well-tuned noise models. Working on SE(2) also reinforced my appreciation for the subtleties of state estimation on manifolds.
