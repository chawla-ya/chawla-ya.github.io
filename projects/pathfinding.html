Overview

Built a two-stage path planning pipeline that combines Probabilistic Roadmap (PRM) construction with A* graph search. The PRM builds a reusable graph of collision-free configurations sampled from the workspace, and A* efficiently finds the shortest path through this graph from start to goal.

Problem

Path planning in continuous configuration spaces with obstacles requires balancing completeness, optimality, and computational efficiency. Grid-based methods scale poorly to high dimensions, while pure sampling methods like RRT don't guarantee shortest paths. PRM + A* provides a practical middle ground: a precomputed roadmap that supports fast, near-optimal queries for multiple start-goal pairs.

Approach

The PRM construction phase uniformly samples configurations in free space, connects nearby samples with collision-free edges (verified via line-of-sight checks), and builds a graph data structure. The sampling density and connection radius are tuned to balance roadmap coverage with construction time.

Given a start and goal configuration, both are connected to the existing roadmap, and A* search with a Euclidean distance heuristic finds the optimal path through the graph. The heuristic is admissible and consistent, guaranteeing A* returns the shortest graph path.

Key Features

PRM Construction

Efficient roadmap building with k-nearest neighbor connections and collision-free edge validation.

A* Search

Optimal graph search with admissible heuristic guaranteeing shortest-path solutions.

Multi-Query Support

Precomputed roadmap enables fast planning for multiple start-goal pairs without rebuilding.

Configurable Parameters

Tunable sample count, connection radius, and collision resolution for different environments.

Results

The planner successfully found collision-free paths through cluttered 2D environments with varying obstacle densities. Path quality improved with increasing sample count, converging toward the true shortest path. The multi-query capability demonstrated significant speedup over single-query methods for repeated planning in static environments.

What I Learned

This project strengthened my understanding of sampling-based planning algorithms, graph search optimality guarantees, and the engineering tradeoffs in building practical planning systems. Implementing both the roadmap construction and search phases from scratch gave me insight into how these components interact in real robotics applications.
