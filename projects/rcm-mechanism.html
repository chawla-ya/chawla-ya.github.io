Overview

Designed and analyzed a 2-DOF Remote Center of Motion (RCM) mechanism for ear and facial microsurgery, where extremely fine precision is required in tight workspaces. The mechanism is based on a modified agile eye architecture ‚Äî a robot with revolute joints whose axes intersect at a single virtual point. The end effector pivots about this remote fixed point (the RCM) without any physical joint at that location, which is critical for minimally invasive procedures where the instrument passes through a trocar port or incision.

Mechanism Architecture

The mechanism uses a double-parallelogram linkage that maintains two sets of parallel links at all times. This parallelism constraint ensures that motion of either input is mirrored as an opposite motion on the output side, keeping the pivot point stationary. The key components are:

  1. Revolute Platform ‚Äî provides the first revolute input Œ∏‚ÇÅ
  2. Revolute Base ‚Äî ground / fixed frame of the mechanism
  3. Primary Input Link ‚Äî provides the second revolute input Œ∏‚ÇÇ
  4. Secondary Input Link ‚Äî parallel with primary input link and end effector
  5. Primary & Secondary Support Links ‚Äî parallel with each other, forming the parallelogram with both input links
  6. End Effector ‚Äî houses the medical trocar or surgical instrument, pivots about the RCM

The double-parallelogram geometry allows the end effector to pitch up/down and yaw through the fixed RCM point without translating the pivot itself ‚Äî critical for avoiding tissue damage in surgery or maintaining a fixed optical focus point in camera systems.

Forward Kinematics

Due to the 1:1 input-output mapping of the parallelogram linkage, forward kinematics can be solved by inspection. The homogeneous transformation for any mechanism motion is a pure rotation matrix (R appended with a zero translation column), since the end effector undergoes pure rotation about the RCM with no translation. The full transformation T_SB from the world frame {S} to the body frame {B} does include a translation component accounting for the offset of the end effector frame from the world origin, with magnitude dependent on the link lengths.

Proposed Design Improvements

The current 2-DOF design was analyzed for potential upgrades. Adding a belt/screw-driven cylindrical joint on the end effector would provide both prismatic actuation along the instrument axis and rotation about the body-centric Y axis ‚Äî adding 2 DOF while minimizing sprung mass for faster, more precise control. Replacing the revolute base joint with a spherical joint would add curvilinear Z-axis translation, though for surgical applications this additional freedom may be less desirable from a safety standpoint.
